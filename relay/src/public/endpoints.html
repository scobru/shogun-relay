GunDB Core

GET /gun Public

WebSocket endpoint for Gun clients. Connect using Gun.js library. 

// JavaScript
const gun = Gun('ws://localhost:8765/gun');

// Or with multiple peers
const gun = Gun({
  peers: ['ws://localhost:8765/gun']
});
GET /api/v1/system/node/* Optional

Inspect GunDB nodes via REST API. Use wildcard path to navigate the graph.

// Get a node
curl http://localhost:8765/api/v1/system/node/users/alice

// Response
{
  "success": true,
  "data": {
    "name": "Alice",
    "age": 30
  }
}
POST /api/v1/system/node/* Admin

Update or create GunDB nodes via REST API.

curl -X POST http://localhost:8765/api/v1/system/node/users/alice \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD" \
  -H "Content-Type: application/json" \
  -d '{"name": "Alice", "age": 30}'
DELETE /api/v1/system/node/* Admin

Delete GunDB nodes via REST API.

curl -X DELETE http://localhost:8765/api/v1/system/node/users/alice \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD"

IPFS Management

POST /api/v1/ipfs/upload Admin

Upload files to IPFS. Supports multipart/form-data with optional encryption.

// Using FormData
const formData = new FormData();
formData.append('file', fileBlob);
formData.append('encrypt', 'true'); // Optional

fetch('http://localhost:8765/api/v1/ipfs/upload', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_ADMIN_PASSWORD'
  },
  body: formData
})
.then(res => res.json())
.then(data => console.log(data));

// Response
{
  "success": true,
  "data": {
    "hash": "QmXxxx...",
    "size": 12345,
    "relayUrl": "http://localhost:8765/api/v1/ipfs/cat/QmXxxx..."
  }
}
GET /api/v1/ipfs/cat/:cid Optional

Stream IPFS content (aligned with Kubo's /api/v0/cat).

curl http://localhost:8765/api/v1/ipfs/cat/QmXxxx...
GET /api/v1/ipfs/cat/:cid/json Optional

Get IPFS content as JSON. Automatically parses JSON content.

curl http://localhost:8765/api/v1/ipfs/cat/QmXxxx.../json

// Response
{
  "success": true,
  "cid": "QmXxxx...",
  "data": { "key": "value" }
}
GET /api/v1/ipfs/cat/:cid/decrypt Optional

Get and decrypt SEA-encrypted IPFS content.

// Encrypted content with decryption token
curl "http://localhost:8765/api/v1/ipfs/cat/QmXxxx.../decrypt?token=YOUR_ENCRYPTION_KEY"
POST /api/v1/ipfs/pin/add Admin

Pin content to IPFS (aligned with Kubo's /api/v0/pin/add).

curl -X POST http://localhost:8765/api/v1/ipfs/pin/add \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD" \
  -H "Content-Type: application/json" \
  -d '{"cid": "QmXxxx..."}'

// Response
{
  "success": true,
  "message": "CID pinned successfully",
  "result": { "Pins": ["QmXxxx..."] }
}
POST /api/v1/ipfs/pin/rm Admin

Remove a pin from IPFS (aligned with Kubo's /api/v0/pin/rm).

curl -X POST http://localhost:8765/api/v1/ipfs/pin/rm \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD" \
  -H "Content-Type: application/json" \
  -d '{"cid": "QmXxxx..."}'
GET /api/v1/ipfs/pin/ls Admin

List all pinned content (aligned with Kubo's /api/v0/pin/ls).

curl http://localhost:8765/api/v1/ipfs/pin/ls \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD"

// Response
{
  "success": true,
  "pins": { "QmXxxx...": { "Type": "recursive" } },
  "count": 1
}
GET /api/v1/ipfs/status Public

Check IPFS node connectivity and status.

curl http://localhost:8765/api/v1/ipfs/status

// Response
{
  "success": true,
  "data": {
    "connected": true,
    "version": "0.24.0",
    "peers": 5
  }
}
GET /api/v1/ipfs/repo/stat Admin

Get IPFS repository statistics (size, objects, etc.).

curl http://localhost:8765/api/v1/ipfs/repo/stat \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD"
POST /api/v1/ipfs/repo/gc Admin

Run garbage collection to remove unpinned content from IPFS.

curl -X POST http://localhost:8765/api/v1/ipfs/repo/gc \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD" \
  -H "Content-Type: application/json" \
  -d '{}'
GET /api/v1/ipfs/version Public

Get IPFS version information.

curl http://localhost:8765/api/v1/ipfs/version
GET /ipfs/:cid Public

IPFS Gateway proxy. Access content directly via CID.

// Direct gateway access
curl http://localhost:8765/ipfs/QmXxxx...

// Or open in browser
open http://localhost:8765/ipfs/QmXxxx...
GET /ipns/:name Public

IPNS Gateway proxy. Resolve IPNS names.

curl http://localhost:8765/ipns/k51qzi5uqu5...
POST /api/v0/* Admin

Raw Kubo IPFS API proxy. Forwards requests to the IPFS node API.

// Example: Get IPFS ID
curl -X POST http://localhost:8765/api/v0/id \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD"

// Example: Add file
curl -X POST http://localhost:8765/api/v0/add \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD" \
  -F "file=@/path/to/file"

System & Debug

GET /health Public

Simple health check endpoint.

curl http://localhost:8765/health

// Response
{
  "success": true,
  "message": "Shogun Relay API is running",
  "data": {

    "version": "1.0.0",
    "uptime": 3600
  }
}

Check Holster Relay service status and configuration.

curl http://localhost:8765/holster-status

// Response
{
  "success": true,
  "status": "active",
  "service": "holster-relay",
  "config": {
    "port": 8766,
    "host": "0.0.0.0",
    "storageEnabled": true,
    "storagePath": "./holster-data",
    "maxConnections": 100
  },
  "timestamp": 1704067200000

GET /api/v1/system/health Public

Detailed health report with system information.

curl http://localhost:8765/api/v1/system/health
GET /api/v1/system/relay-info Public

Get relay server information.

curl http://localhost:8765/api/v1/system/relay-info
GET /api/v1/system/stats Public

Get system statistics and metrics.

curl http://localhost:8765/api/v1/system/stats
GET /api/v1/system/alldata Admin

Dump all data from the 'shogun' namespace in GunDB.

curl http://localhost:8765/api/v1/system/alldata \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD"
GET /api/v1/system/logs Admin

Get system logs.

curl http://localhost:8765/api/v1/system/logs \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD"
DELETE /api/v1/system/logs Admin

Clear system logs.

curl -X DELETE http://localhost:8765/api/v1/system/logs \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD"
GET /api/v1/system/peers Public

List all GunDB peers.

curl http://localhost:8765/api/v1/system/peers
POST /api/v1/system/peers/add Admin

Add a GunDB peer.

curl -X POST http://localhost:8765/api/v1/system/peers/add \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD" \
  -H "Content-Type: application/json" \
  -d '{"peer": "wss://another-relay.example.com/gun"}'

Services

Get status of all services (IPFS, Relay, etc.).

curl http://localhost:8765/api/v1/services/status
POST /api/v1/services/:service/restart Admin

Restart a specific service (stub endpoint).

curl -X POST http://localhost:8765/api/v1/services/ipfs/restart \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD"

User Uploads

GET /api/v1/user-uploads/system-hashes Public

Get system hash registry (protected hashes that won't be garbage collected).

curl http://localhost:8765/api/v1/user-uploads/system-hashes
GET /api/v1/user-uploads/system-hashes-map Public

Get system hash registry as a map for quick lookups.

curl http://localhost:8765/api/v1/user-uploads/system-hashes-map
GET /api/v1/user-uploads/:identifier Public

Get all uploads for a specific user identifier.

curl http://localhost:8765/api/v1/user-uploads/user123
GET /api/v1/user-uploads/:identifier/:hash Public

Get specific upload information for a user.

curl http://localhost:8765/api/v1/user-uploads/user123/QmXxxx...
POST /api/v1/user-uploads/save-system-hash Admin

Save a hash to the system registry to protect it from garbage collection.

curl -X POST http://localhost:8765/api/v1/user-uploads/save-system-hash \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD" \
  -H "Content-Type: application/json" \
  -d '{"hash": "QmXxxx..."}'
DELETE /api/v1/user-uploads/remove-system-hash/:hash Admin

Remove a hash from the system registry.

curl -X DELETE http://localhost:8765/api/v1/user-uploads/remove-system-hash/QmXxxx... \
  -H "Authorization: Bearer YOUR_ADMIN_PASSWORD"
