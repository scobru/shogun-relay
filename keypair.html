Shogun Key Pair Generator

Important Security Notice: The key pair generated here is meant for development and testing purposes. In production environments, key pairs should be generated securely on the client device. Never share your private keys!

This tool generates a cryptographic key pair that can be used with Gun.js and Shogun applications. The key pair consists of a public key (pub) and a private key (priv).

How to Use:

  1. Click the "Generate New Key Pair" button to create a new random key pair.
  2. Click "Download Key Pair" to save the JSON file containing your keys.
  3. You can use this key pair for authentication with Gun.js or Shogun applications.
  4. To authenticate with your key pair, you can use: 
    gun.user().auth(keypair)

API Usage Example:

// Authenticate with key pair
fetch('http://localhost:8765/api/auth/verify-cert', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ 
    certificate: yourKeyPair 
  })
})
.then(response => response.json())
.then(data => console.log(data));
